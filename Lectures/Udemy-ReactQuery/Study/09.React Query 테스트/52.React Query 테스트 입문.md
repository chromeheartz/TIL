#### ✅ 테스팅에 대한 고려 사항

- 일반적으로 테스팅 라이브러리에서 권장하는 방법을 동일하게 따르는 것이 좋다.
  - 즉, `내부 진행 상황` 보다는 `사용자가 소프트웨어` 와 상호 작용하는 방법을 다룬다
- `테스팅 쿼리, 변이, 테스팅 커스텀 훅` 의 예시를 정리할 것이다.
  - `React Query` 를 포함하는 테스트 부분들에 집중
