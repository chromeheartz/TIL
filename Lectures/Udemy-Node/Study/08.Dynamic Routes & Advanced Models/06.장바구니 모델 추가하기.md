### 📌 장바구니 모델 추가하기

현재는 장바구니가 없다. 이제 새로운 모델을 추가하자
`modes/cart.js`

#### cart.js

```js
const fs = require("fs");
const path = require("path");

const p = path.join(
  path.dirname(process.mainModule.filename),
  "data",
  "cart.json",
);

module.exports = class Cart {
  static addProduct(id, productPrice) {
    // Fetch the previous cart
    fs.readFile(p, (err, fileContent) => {
      let cart = { products: [], totalPrice: 0 };
      if (!err) {
        cart = JSON.parse(fileContent);
      }
      // Analyze the cart => Find existing product
      const existingProductIndex = cart.products.findIndex(
        (prod) => prod.id === id,
      );
      const existingProduct = cart.products[existingProductIndex];
      let updatedProduct;
      // Add new product/ increase quantity
      if (existingProduct) {
        updatedProduct = { ...existingProduct };
        updatedProduct.qty = updatedProduct.qty + 1;
        cart.products = [...cart.products];
        cart.products[existingProductIndex] = updatedProduct;
      } else {
        updatedProduct = { id: id, qty: 1 };
        cart.products = [...cart.products, updatedProduct];
      }
      cart.totalPrice = cart.totalPrice + +productPrice;
      fs.writeFile(p, JSON.stringify(cart), (err) => {
        console.log(err);
      });
    });
  }
};
```

클래스를 `exports` 하고 이름은 `Cart` 로 정해준다.
먼저 구현하기위해 `constroctur` 를 만들어줄것이다.

```js
constructor() {
	this.products = [];
  	this.totalPrice = 0;
}
```

`cart` 를 만들고나면 여기에 `products` 속성을 추가할것이다.
이는 하나의 배열이며 이 안에 ID나 중요한 정보가 담긴 객체가 들어가며 제품의 수량도 들어간다.

> ✅ 지금 장바구니에 필요한 것은 제품을 `추가 / 제거` 할 수 있는 방법이다. 문제는 장바구니 자체가 계속해서 다시 생성할 객체가 아니라는 점이다. 새로운 제품을 추가할 때마다 `새로운 cart` 를 만들지는 않고, 전체 애플리케이션 내에 `하나의 장바구니` 를 만들어 관리하도록 해야하니 다른 접근법을 사용해보자.

#### 다른 접근법

`static` 메소드를 추가해서 `addProduct` 같은 식으로 여기에 추가하길 원하는 제품의 ID가 들어가게 된다. `static addProduct(id, productPrice) {...}` 같은 형식이 된다.
추가하길 원하는 `id` 가 들어가고, 해당 제품이 이미 있는지 확인해서 기존 제품을 찾고 새로운 제품을 추가하거나 수량을 증가시키는 방식이 될것이다.

먼저 `fs, path` 를 import한다.
기존에 `product.js` 에서 `join` 메소드를 사용해서 경로를 만들었는데, 해당 구문을 가져온다.

그리고 제품을 추가하는 부분에 대해서는 파일시스템을 사용해 경로에 있는 `cart.json` 파일을 읽어서 오류 / 콘텐츠를 회신하는 콜백을 만든다.

`fs.readFile(p, (err, fileContent) => {...})`
오류가 발생하지 않으면 반대 논리에 따라 기존 cart가 존재함을 알 수 있으니, 이 경우에는 파싱을 해주면 된다.

이후에 `cart` 를 분석해서 추가하려는 제품이 이미 있는지 확인을 해본다.
`findIndex` 메소드를 이용해서 찾아보면 된다. 만약 기존 제품이 있다면 간단히 수량만 늘려준다.

여기에 저장되는 각 product 객체가 product 모델에 데이터가 있는 단순한 product 객체가 아니라 추가적인 수량 필드도 가진다고 가정을 해보자.

또한 만약 기존 제품이 있다면 새로운 제품을 만들어준다.
`스프레드 연산자` 를 사용해서 모든 속성을 가지고가서 새로운 객체에 추가해주면 된다.

그리고 마지막으로는 `장바구니의 가격` 을 갱신해주어야 한다.
가격만큼 증가해줄 수 있도록 인수로 받고, `cart` 를 `wrtieFile` 메소드로 파일에 저장해주면 된다.

가격 부분에서`문자열` 로 저장되었기 때문에 이를 해결해야 하는데
`cart.js` 에서 추출하는 가격은 `product` 모델에 문자열 형태로 저장되어있어, 가격을 다룰 때는 `+` 기호를 붙여서 문자열을 숫자로 변환해주면 된다.
