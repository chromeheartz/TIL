### 📌 장바구니에 항목 표시하기

이제 `cart.ejs` 에서 장바구니 항목을 보여주자.

```html
<%- include('../includes/head.ejs') %>
    </head>

    <body>
        <%- include('../includes/navigation.ejs') %>
        <main>
            <% if (products.length > 0) { %>
                <ul>
                    <% products.forEach(p => { %>
                        <li>
                            <p><%= p.productData.title %> (<%= p.qty %>)</p>
                            <form action="/cart-delete-item" method="POST">
                                <input type="hidden" value="<%= p.productData.id %>" name="productId">
                                <button class="btn" type="submit">Delete</button>
                            </form>
                        </li>
                    <% }) %>
                </ul>
            <% } else { %>
                <h1>No Products in Cart!</h1>
            <% } %>
        </main>
        <%- include('../includes/end.ejs') %>
```

제품이 있는지 확인을 하고 제품을 보여주도록 한다.
`cart` 를 불러왔던 `shop controller` 에서 모든 제품을 가져올 준비를 해야한다.

```js
// cart.js
static getCart(cb) {
    fs.readFile(p, (err, fileContent) => {
      const cart = JSON.parse(fileContent);
      if (err) {
        cb(null);
      } else {
        cb(cart);
      }
    });
  }
```

새로운 `static` 함수로 `getProducts` 를 추가하고 파일에 접근해서 제품 ID를 반환하도록 한다. 따라서 제품 정보를 받은 후에 호출하는 콜백이 있어야 한다. 파일을 읽기 위해 `readFile` 함수를 복사해온다. 콜백으로 장바구니 전체를 반환할것이다.

장바구니가 아직 없으면 실패할 수 있으니 `if(err)` 을 이용해 오류가 생기는지 확인
오류가 없다면 유효한 값과 함께 장바구니 전체를 콜백에서 반환하도록 한다.

```js
// controllers/shop.js
exports.getCart = (req, res, next) => {
  Cart.getCart((cart) => {
    Product.fetchAll((products) => {
      const cartProducts = [];
      for (product of products) {
        const cartProductData = cart.products.find(
          (prod) => prod.id === product.id,
        );
        if (cartProductData) {
          cartProducts.push({ productData: product, qty: cartProductData.qty });
        }
      }
      res.render("shop/cart", {
        path: "/cart",
        pageTitle: "Your Cart",
        products: cartProducts,
      });
    });
  });
};
```

`shop.js` 컨트롤러의 `getCart` 에 이제 `Cart.getCart` 를 추가해주고 모델에 방금 추가한 콜백 함수를 전달한다. 또 함수 안에서 뷰를 렌더링한다.
이때 `cart` 와 더불어 제품에 관한 정보도 필요하기 때문에 `Product 모델` 에 콜백을 추가해준다.

`for(poroduct of products)` 를 하면 모든 제품에 루핑이 적용되어서 특정 제품이 장바구니에 있는지 확인이 된다. `product id` 가 있기 때문에 가능하다.
많은 양의 데이터를 저장했다면 코드를 좀 더 발전시켜야 하지만 어차피 이 파일에는 데이터를 많이 저장하지 않았기 때문에 가능하다.
이후에는 `데이터베이스` 를 사용.

만약 ID를 찾으면 바로 제품이 있다는 뜻이므로 `const` 를 생성해서 `cartProducts` 배열을 만들고 찾은 제품을 넣도록 한다.`qty` 수량필드도 까먹지 않고 넣어주어야 한다.
