### 📌 제품 상세 보기 렌더링

이제 뷰를 추가하거나 기존 뷰에 논리를 추가해보자.
이를 위해 `orders.ejs` 와 같은 다른 뷰의 코드를 가져와서 `product-detail.ejs` 에 붙여넣자.

```html
<%- include('../includes/head.ejs') %>
    </head>

    <body>
        <%- include('../includes/navigation.ejs') %>
        <main class="centered">
            <h1><%= product.title %></h1>
            <hr>
            <div>
                <img src="<%= product.imageUrl %>" alt="<%= product.title %>">
            </div>
            <h2><%= product.price %></h2>
            <p><%= product.description %></p>
            <form action="/cart" method="post">
                <button class="btn" type="submit">Add to Cart</button>
            </form>
        </main>
        <%- include('../includes/end.ejs') %>
```

이런식으로 뷰를 만들어주었다. 이제, 렌더링을 위해 컨트롤러에서 입력해준다.

```js
exports.getProduct = (req, res, next) => {
  const prodId = req.params.productId;
  Product.findById(prodId, (product) => {
    res.render("shop/product-detail", {
      product: product,
      pageTitle: product.title,
      path: "/products",
    });
  });
};
```

`res.render` 에 파일명을 입력해주고, 정보를 입력해야 한다.
이는 자바스크립트 객체를 통해 가능한데, 상품 속성을 입력해주어야 한다.
뷰에서 필요한 속성들을 `render` 메소드 사용시에 보내주면 된다.

`path` 속성을 안보내주면 경로가 없다고나오는데,
네비게이션에서 이를 사용해서 어떤 경로가 `active` 상태인지 확인하고 있기 때문에 `path` 또한 같이 보내주어야 한다.
