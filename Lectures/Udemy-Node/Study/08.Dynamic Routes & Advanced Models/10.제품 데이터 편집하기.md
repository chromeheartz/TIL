### 📌 제품 데이터 편집

먼저 `Product Model` 작업을 하자.
`save` 메소드를 사용해서 새 제품을 생성하고 있는데, 이미 확보하고 있는 기존의 제품을 `업데이트` 할 때도 사용하게 하고싶다.
이를 위해 이미 `id` 가 있는지 확인하고 새 제품을 생성할 때 `id` 도 받아오도록 한다.

```js
// models/product.js
module.exports = class Product {
  constructor(id, title, imageUrl, description, price) {
    this.id = id;
    this.title = title;
    this.imageUrl = imageUrl;
    this.description = description;
    this.price = price;
  }

  save() {
    getProductsFromFile(products => {
      if (this.id) {
        const existingProductIndex = products.findIndex(
          prod => prod.id === this.id
        );
        const updatedProducts = [...products];
        updatedProducts[existingProductIndex] = this;
        fs.writeFile(p, JSON.stringify(updatedProducts), err => {
          console.log(err);
        });
      } else {
        this.id = Math.random().toString();
        products.push(this);
        fs.writeFile(p, JSON.stringify(products), err => {
          console.log(err);
        });
      }
    });
  }
```

`id` 를 기준으로 제품을 생성할것인지 업데이트를 할것인지 분기를 나누어준다.
이 후 `postAddProduct` 메소드도 더 수정해주어야 한다.
`const product = new Product(null, title, imageUrl, description, price);`

해당 구문으로 `null` 인 경우를 넣어주어야 새로운 제품 생성모드로 진입할 수 있게 된다.

이제 edit 메소드를 추가해주자.

```js
exports.postEditProduct = (req, res, next) => {
  const prodId = req.body.productId;
  const updatedTitle = req.body.title;
  const updatedPrice = req.body.price;
  const updatedImageUrl = req.body.imageUrl;
  const updatedDesc = req.body.description;
  const updatedProduct = new Product(
    prodId,
    updatedTitle,
    updatedImageUrl,
    updatedDesc,
    updatedPrice,
  );
  updatedProduct.save();
  res.redirect("/admin/products");
};
```

먼저 제품에 대한 정보를 가져와야 한다. 각각 `req.body` 에서 추출해준다.
