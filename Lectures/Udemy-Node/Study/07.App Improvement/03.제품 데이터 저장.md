### 📌 제품 데이터 저장

논리를 추가할 시간이다. 어디서부터 시작해야할까?
`제목` 만이 아니라 `이미지, 가격 설명` 텍스트도 추가할 수 있도록 하기 위해 일단 `모델` 을 먼저 들여다봐야 한다.
모델에서 `title` 만을 다루는 것이 아니라 `title` 로 새로운 제품의 초기 설정을 할 수 있도록 하자.
이미지url을 추가해주고, description, price또한 추가해준다.

```js
// product.js
  constructor(title, imageUrl, description, price) {
    this.title = title;
    this.imageUrl = imageUrl;
    this.description = description;
    this.price = price;
  }
```

이렇게 모든 데이터를 속성에 저장해준다. 이렇게 하면 파일에 추가하는 제품에 이러한 데이터들이 포함되게 되고, 불러오는 데이터나 제품도 마찬가지이다.

이제 `Add Product view` 만 수정하면 된다.
추가 데이터를 불러오기 위해 input을 추가해주고 이를 위해 `form-control` 을 복사해서 하나를 더 추가한다.

```html
<%- include('../includes/head.ejs') %>
    <link rel="stylesheet" href="/css/forms.css">
    <link rel="stylesheet" href="/css/product.css">
</head>

<body>
   <%- include('../includes/navigation.ejs') %>

    <main>
        <form class="product-form" action="/admin/add-product" method="POST">
            <div class="form-control">
                <label for="title">Title</label>
                <input type="text" name="title" id="title">
            </div>
            <div class="form-control">
                <label for="imageUrl">Image URL</label>
                <input type="text" name="imageUrl" id="imageUrl">
            </div>
            <div class="form-control">
                <label for="price">Price</label>
                <input type="number" name="price" id="price" step="0.01">
            </div>
            <div class="form-control">
                <label for="description">Description</label>
                <textarea name="description" id="description" rows="5"></textarea>
            </div>

            <button class="btn" type="submit">Add Product</button>
        </form>
    </main>
<%- include('../includes/end.ejs') %>
```

`imageUrl / description / price` 를 추가하기 위한 인풋을 만들어주어야 한다.
각각 id를 맞추어서 넣어주면 된다.
이제 input 필드에 부여한 이름으로 이 모든 데이터를 추출할 수 있다.
이렇게 하고 컨트롤러인 `postAddProduct` 가 있는 `admin controller` 로 돌아가서 새 제품을 생성하고, 데이터들을 추출한다.
좀 더 가독성을 높게 하기 위해서 구조화된 방법으로 수정.

```js
exports.postAddProduct = (req, res, next) => {
  const title = req.body.title;
  const imageUrl = req.body.imageUrl;
  const price = req.body.price;
  const description = req.body.description;
  const product = new Product(title, imageUrl, description, price);
  product.save();
  res.redirect("/");
};
```

여기서 값을 추출해서 상수에 저장하게 된다.
이렇게 데이터를 추출하면 전달할 수 있는데, 생성자에서 정의한 순서에 주의해야한다.
이제 이 모든 데이터로 제품을 생성하면 된다.
