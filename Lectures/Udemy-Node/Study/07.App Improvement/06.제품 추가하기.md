### 📌 제품 추가하기

`shop` 에 간단한 뷰 하나를 추가하자.
`orders.ejs` 라우트로, 이는 주문을 표시해주는 역할을 하게 된다.
아직은 주문이 없으므로 `cart.ejs` 의 내용을 가져와서 내비게이션에서만 표시되도록 하자.
물론 사이에 무언가를 표시할 수도 있다.

````html
<%- include('../includes/head.ejs') %>
  </head>

<body>
  <%- include('../includes/navigation.ejs') %>
    <main>
      <h1>Nothing there!</h1>
    </main>
    <%- include('../includes/end.ejs') %>
      ```

`orders` 라는 경로를 `nav` 에도 추가해준다.
그리고 `routes/shop.js` 에는 `/orders` 로 가게 해주는 새로운 라우트를 추가해야 한다.
이는 `shop.js` 컨트롤러에 새로운 액션이 필요함을 뜻함.

따라서 `getcart` 를 복사해서 만들어준다.

```js
exports.getOrders = (req, res, next) => {
  res.render('shop/orders', {
    path: '/orders',
    pageTitle: 'Your Orders'
  });
};
````

일단 이렇게 추가하고 이후 모듈에서 진행해보자.
이후에 실제로 `Express` 라우터를 활용해서 동적데이터나 `url의 일반 데이터` 를 처리하는 방법을 알아볼것이다.
