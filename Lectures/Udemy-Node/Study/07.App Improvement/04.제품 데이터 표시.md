### 📌 데이터 표시

이제 이 데이터들을 표시하고 더미데이터를 표시하지 않도록 해보자.
일단 `product-list.ejs` 파일로 가서 모든 제품이 표시되는 뷰 중 하나인데, 이미 실제 제품 제목을 출력하고 있지만 그게 전부이다.
이미지 소스 역시 교체해야 한다.

```html
<!-- product-list.ejs -->
<%- include('../includes/head.ejs') %>
    <link rel="stylesheet" href="/css/product.css">
</head>

<body>
    <%- include('../includes/navigation.ejs') %>

    <main>
        <% if (prods.length > 0) { %>
            <div class="grid">
                <% for (let product of prods) { %>
                    <article class="card product-item">
                        <header class="card__header">
                            <h1 class="product__title"><%= product.title %></h1>
                        </header>
                        <div class="card__image">
                            <img src="<%= product.imageUrl %>"
                                alt="<%= product.title %>">
                        </div>
                        <div class="card__content">
                            <h2 class="product__price">$<%= product.price %></h2>
                            <p class="product__description"><%= product.description %></p>
                        </div>
                        <div class="card__actions">
                            <button class="btn">Add to Cart</button>
                        </div>
                    </article>
                <% } %>
            </div>
        <% } else { %>
            <h1>No Products Found!</h1>
        <% } %>
    </main>
<%- include('../includes/end.ejs') %>
```

`ejs` 표현을 사용해서 `imageUrl` 을 출력하면 유효한 `url` 문자열이 되므로 표시할 수 있게 된다.
대체 텍스트로는 `title` 을 사용.
가격은 항상 `$`가 되어야 해서 `$` 를 붙여서 `product.price` 를 보여준다.
기존에 작업했던 `product.json` 과는 객체구조가 다르니 해당 부분도 수정해주고 진행하도록 한다.

이제 제품을 표시하는데에는 문제가 없는데 `편집 / 삭제` 도 가능하도록 구현해보자.
