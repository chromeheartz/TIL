### 📌 레이아웃 추가

일단 기존에 있던 html들을 pug로 바꾸었다.
이제 모든 파일에서 계속 기본 구조를 반복하고 있는데 이걸 수동으로 진행한다는건 당연히 좀 성가신일이 될것이다. 예를 들어 `기본 설정 / main.css` 및 헤더의 불러오기는 각 페이지의 일부에 해당하며 따라서 이 설정 내지는 불러오기를 항상 수동으로 반복해야 한다면 좀 번거로울 수 있다.

이때 `레이아웃` 이란건 생성할 수 있다.
`views` 에 하위 폴더를 생성하고 `layouts` 라고 이름을 붙인 뒤 `main-layout.pug` 를 추가해보자.

```html
<!-- main-layout.pug -->
<!DOCTYPE html>
html(lang="en") head meta(charset="UTF-8") meta(name="viewport",
content="width=device-width, initial-scale=1.0")
meta(http-equiv="X-UA-Compatible", content="ie=edge") title #{pageTitle}
link(rel="stylesheet", href="/css/main.css") block styles body
header.main-header nav.main-header__nav ul.main-header__item-list
li.main-header__item a(href="/", class=(path === '/' ? 'active' : '')) Shop
li.main-header__item a(href="/admin/add-product", class=(path ===
'/admin/add-product' ? 'active' : '')) Add Product block content

<!-- add-product.pug -->
extends layouts/main-layout.pug block styles link(rel="stylesheet",
href="/css/forms.css") link(rel="stylesheet", href="/css/product.css") block
content main form.product-form(action="/admin/add-product", method="POST")
.form-control label(for="title") Title input(type="text", name="title")#title
button.btn(type="submit") Add Product
```

해당 형태가 될것이다.
레이아웃을 재사용하려면 `block` 이라는 키워드를 추가하고 `content` 를 넣어주면 된다. 추가로
`block styles` 같은 이름을 정의해서 넣어줄 수 있다.

이후에 `extends` 키워드를 추가하여 레이아웃을 확장할 수 있고, 지정해주기만 하면 된다.
`block content / block styles` 둘 다 커스텀으로 각각 개별 파일에서 설정할 수 있다.

### 📌 Pug 템플릿 완성하기

어느 페이지에 있는지에 따라 올바른 링크에 활성화 클래스를 설정해야 했다.

```node
// admin.js
router.get("/add-product", (req, res, next) => {
  res.render("add-product", {
    pageTitle: "Add Product",
    path: "/admin/add-product",
  });
});

// shop.js
router.get("/", (req, res, next) => {
  const products = adminData.products;
  res.render("shop", { prods: products, pageTitle: "Shop", path: "/" });
});
```

`path` 를 `view` 로 넘기면 어느 것에 대한 경로가 로딩된 것인지 알아낼 수 있다.
메인 레이아웃에 이 경로를 이용해 작업을 할 수 있다.

```html
ul.main-header__item-list li.main-header__item a(href="/", class=(path === '/' ?
'active' : '')) Shop li.main-header__item a(href="/admin/add-product",
class=(path === '/admin/add-product' ? 'active' : '')) Add Product
```

`삼항 연산자` 를 사용해서 `class` 에 `active` 효과도 줄 수 있다.
