## Workflow & Debugging

### 03. 전역 기능 vs 코어 모듈 vs 제3자 모듈

---

기본적으로 이 3가지는 이렇게 구분할 수 있다.

- `전역 기능` : const / function 같은 키워드 및 process 등의 `전역 객체`
- `코어 Node.js 모듈` : 파일 시스엠 모듈("fs"), 경로 모듈 ("path"), Http 모듈("http") 등
- `제 3자 모듈` : npm install을 통해 어떤 종류의 기능도 설치 가능

**전역 기능** 은 **항상 사용 가능**하며 사용하기 위해 파일에 임포트할 필요가 없다.
코어 Node.js 모듈은 설치 하지 않아도 되기 때문에 `npm install` 이 필요하지 않지만, 관련된 기능을 사용하려면 임포트 해야한다.

예시
`const fs = require('fs')`
"fs"모듈에서 내보낸 `fs` 객체를 사용할 수 있게 된다.
프로젝트 폴더에 `npm install` 을 실행해 제3자 모듈을 설치하고 임포트한다.

```js
// Terminal 또는 명령 프롬프트에서는

npm install --save express-session
// app.js 같은 코드 파일에서는

const sessions = require('express-session');
```
